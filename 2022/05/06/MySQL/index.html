    索引

    索引简介

    索引是一个单独、存储在磁盘上的数据结构，它们包含着对数据表里所有记录的引用指针。
    索引的存储类型有两种
  • BTREE
  • HASH
    存储引擎MyISAM和InnoDB只支持BTREE,MEMORY/HEAP引擎支持HASH和BTREE

    索引分类

  • 普通索引和唯一索引
  • 单列索引和组合索引
  • 全文索引
  • 空间索引

    索引设计原则

  • 索引并非越多越好，一个表中有大量索引，不仅占用磁盘空间，而且会影响INSERT,DELETE,UPDATE等语句的性能，因为当表中的数据更改时，索引也会进行调整和更新。
  • 避免对经常更新的表进行过多的索引，并且索引中的列尽可能少。面对经常查询的字段应该创建索引，弹药避免添加不必要的字段
  • 数据量小的表最好不要使用索引，由于数据较少，查询花费的时间可能比遍历索引的时间还要短，索引可能不会产生优化效果
  • 在条件表达式中经常用到的不同值较多的列上建立索引，在不同值很少的列上不要创建索引。比如学生表中的性别“男”和“女”
  • 当唯一性是某种数据本身的特征时，指定唯一索引。使用唯一索引需要确保定义的列上的完整性，以提高查询速度。
  • 在频繁仅需排序或分组(即进行group by或order by操作)的列上建立索引，如果待排序的列上有多个，可以在这些列上建立组合索引。

    创建索引

    alter table

    alter table table_name [UNIQUE|FULLTEXT|SPATIAL] [INDEX|KEY] [index_name] (column[length],…) [ASC|DESC]
    alter table book add index bkxxIdx ( comment(50) )

    create Index

    CREATE [UNIQUE|FULLTEXT|SPATIAL] INDEX index_name on table_name (col_name[length],..) [ASC|DESC]
    CREATE INDEX bkxxIdx ON book(comment)

删除索引

ALTER TABLE TABLE_NAME DROP INDEX inde_name;

DROP INDEX index_name on table_name

    create user 'tom'@'localhost' identified by 'mypass';

授权

  • 全局层级 grant all on *.*和revoke all on *.*只授予和撤销全局权限。
  • 数据库层级 grant all db_name.和revoke all db_name.* 只授予和撤销指定数据库权限
  • 表层级 grant all on db_name.table_name 和revoke all on db_name.table_name
  • 列层级
  • 子程序级

收回权限

方式一

revoke all privileges grant, option from 'user'@'localhost' [,'user'@'host' ...]

方式二

revoke priv_type [(columns)] [,priv_type [(columns)]] ... on table1,table2 from 'user'@'host' [, 'user'@'host' ...]

如：
revoke update on *.* from 'testuser'@'localhost';

查看权限

show grants for ‘user‘@’host’;

数据备份

表的导出和导入

导出

select导出文本文件

语句格式
select columnlist from table where condition into outfile 'filename' [options]
options 选项

  • FIELDS TERMINATED BY ‘value’ 设置字符分隔符号
  • FIELDS ENCLOSE BY ‘value’ 设置字段的包围符
  • FIELDS ESCAPED BY ‘value’ 设置如何读取和写入特殊字符串只能为单个字符，即设置转义字符，默认为’\‘
  • LINES STARTING BY ‘VALUES’ 设置每行数据开头的字符，可以为每个或多个字符，默认情况下不使用任何字符
  • LINES TERMINATED BY ‘VALUES’ 设置每行数据结尾的字符，可以为单个字符或多个字符，默认值为’\n’

eg:

mysqldump导出文本文件

语法格式

mysql 导出文本文件

mysql -h host -u user -p --execute="sql查询语句" dbname > filename.txt
–vertical 将每条记录分为多行显示
